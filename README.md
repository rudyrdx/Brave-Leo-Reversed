# Brave-Leo-Reversed
 CLI for Brave's Leo chat using x_brave_key

## Requirements
- Python 3.6+
- [x_brave_key](you can get this from analyzing your leo requests or use this one: qztbjzBqJueQZLFkwTTJrieu8Vw3789u)
- [requests](https://pypi.org/project/requests/)

## Running

```bash
python3 chat.py
```
